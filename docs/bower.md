###What Is Bower
Bower是一个客户端技术的软件包管理器，它可用于搜索、安装和卸载如JavaScript、HTML、CSS之类的网络资源。


    npm install -g bower
    
###bower.json
    
    bower init
    
###使用bower安装jQuery

    bower install jquery
    
    
###使用bower卸载jQuery    
    
    $ bower uninstall jquery
    
###Updating packages    

    bower update
