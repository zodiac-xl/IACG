####环境

1. 使用io.js替换node 
2. 使用npm包Babel

>IO.js Bringing ES6 to the Node Community! 可以欢快的使用ES6语法
>
>Babel is a JavaScript compiler. 支持generation语法 Babel需要全局安装 `npm install babel -g`


###IO.js安装方法

npm  install n -g

>n 为node 和 ioJS版本管理插件

n -h
>n的帮助

n io latest
>安装最新io


n [COMMAND] [args]            
>Uses default env (node) 切换为node

n io [COMMAND]
>Sets env as io 切换为io
